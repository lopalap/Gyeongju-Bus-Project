# 🚌 경주 실시간 버스 정보 서비스 (Gyeongju Bus Live)

## 1. 프로젝트 소개
경주 지역의 버스 도착 정보를 실시간으로 확인하고, 사용자가 버스를 놓치지 않도록 푸시 알림을 제공하는 웹 서비스입니다.

## 2. 주요 기능 및 구현 성과
* **지도 인터페이스 (Kakao Maps API)**: 모든 정류장을 지도에 마커로 표시하고, 클릭 시 도착 정보를 팝업으로 노출합니다.
* **스마트 검색 및 즐겨찾기**: 정류장 이름 검색 기능과 자주 가는 정류장 별표(⭐) 등록 기능을 제공합니다.
* **실시간 카운트다운**: API 데이터를 바탕으로 프론트엔드에서 1초 단위 실시간 시간 감소 기능을 구현했습니다.
* **지능형 알림 시스템 (Web Notification API)**: 사용자가 페이지를 보고 있지 않아도 버스 도착 3분 전 브라우저 푸시 알림을 발송합니다.

## 3. 개발 일정 준수 현황
* **1주차**: 요구사항 분석 및 API 선정 완료
* **2주차**: Node.js 서버 환경 구축 및 DB 설계 완료
* **3주차**: React UI 개발 및 실시간 알림 시스템 신규 구현 완료
* **4주차**: 최종 결과물 점검 및 통합 테스트 진행 중
